# Bella Frutta Daikanyama — マーケティング・ブランド プロジェクト

Instagram のみの状態から、ブランド・Web・LINE へ展開するための基盤ドキュメントです。

**Next.js LP ワイヤー**: [`web/`](../web/)（`cd web && npm run dev` で http://localhost:3000）— 法人向けお問い合わせは [`web/.env.example`](../web/.env.example) を参照。  
各ファイルは「意思決定の記録」と「外注・制作時のブリーフ」として使えます。

| ファイル | 目的 |
|---------|------|
| [01-brand-strategy.md](./01-brand-strategy.md) | ブランドの立ち位置・言葉・トーン・差別化 |
| [02-marketing-strategy.md](./02-marketing-strategy.md) | 目標・ターゲット・チャネル・施策の優先順位 |
| [03-website-project.md](./03-website-project.md) | Web サイトの目的・構成・要件 |
| [04-line-official-project.md](./04-line-official-project.md) | LINE 公式の目的・機能・運用設計 |
| [05-reference-taste-and-budget.md](./05-reference-taste-and-budget.md) | 他店事例・ビジュアル候補（A/B/C）・予算感・非プロ向けの安価交渉の軸 |
| [06-copy-and-messaging-hub.md](./06-copy-and-messaging-hub.md) | コピー検討の集約（商品ストーリー・トーン・チャネル別・手順・チェックリスト） |

**次のアクション（推奨）**  
実店舗の事実（メニュー価格・営業時間・こだわり）を各ファイルの `[要確認]` に追記し、Instagram の世界観と矛盾がないか揃えてください。
