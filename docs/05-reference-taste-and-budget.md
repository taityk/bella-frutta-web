# 参考事例・ビジュアル候補・受注予算感

Bella Frutta のブランド・Web の「方向性を決めるための参照」と、**このスコープを受注したときのおおよその予算感**です。金額は市場の幅が大きいため **目安** として使い、見積は案件ごとに分解してください。

---

## 1. 参考にしやすい事例（カテゴリ別）

## 2. Bella Frutta 向け「テイスト候補」3 案（方向性ラベル）

実店舗の内装・ターゲットと照らして **1 本に絞るか、Instagram と Web で主従を付ける**と運用しやすいです。

### 候補 A — **「代官山・ジュエリー」**（プレミアム・余白・写真主役）

- **コンセプト**: 果物＝宝石。影・質感・1 杯のシルエット。
- **カラー**: オフホワイト、ディープグリーン or バーガンディ、ゴールドは差し色のみ。
- **タイポ**: セリフ or 細めサンセリフ、英字ロゴを主に。
- **Web**: スクロール少なめ、大画像、メニューはタイポで美しく。
- **近い参照**: 千疋屋系の「上質」、Sky High / Why Juice のクリーンさのミックス。

### 候補 B — **「農園マーケット」**（フレッシュ・手書き・温かみ）

- **コンセプト**: 今日つくった一杯、季節の変化が楽しみ。
- **カラー**: 果物そのものの色（いちご・マンゴー・キウイ）を **スウォッチとして整理**し使いすぎない。
- **タイポ**: ラウンドしたサンセリフ＋軽い手書きアクセント。
- **Web**: メニューをカード型、ストーリーページで産地・配合の短文。
- **近い参照**: 山下フルーツ農園のポップさを、代官山向けに **トーンを一段落とした**版。

### 候補 C — **「スタジオ・ボールド」**（記号・若年層・SNS 連続性）

- **コンセプト**: 一目で判別できるマーク色＋短い英語コピー。
- **カラー**: 1 色のシグネチャー（例: コーラル、ビビッドグリーン）＋モノトーン。
- **タイポ**: 太字ジオメトリック、ロゴはシンボル＋ワード。
- **Web**: スクロールで Reels 的なリズム、ブロック分け。
- **近い参照**: Fruit5、The Smoothie Co、Joe & The Juice の「記号色」部分のみ抽出。

## 4. 受注時の予算感（日本市場・目安）

**前提**: 1 店舗・Instagram 既存・戦略ドキュメントはクライアント or 別見積もり、修正回数・撮影・ライティングの有無で **±30〜50%** は普通に出る。

### 4-1. ブランディング（ロゴ〜トーン・簡易ガイド）

| パッケージ | 内容のイメージ | 税抜の目安（円） |
|------------|----------------|------------------|
| ライト | ロゴワンパターン、簡易カラー、用途 2〜3（名刺・SNS アイコン想定） | **15 万〜40 万** |
| スタンダード | ロゴバリエーション、カラー・フォント・写真トーンのミニガイド | **40 万〜100 万** |
| フル | 命名・競合整理・ストーリー・パッケージ・サインまで | **100 万〜250 万＋** |

フリーランス〜小規模デザイン会社では、ロゴ単体が **5 万〜30 万**、会社に戦略込みで **20 万〜80 万** などとされる整理も多い（[reiro ロゴ費用比較](https://reiro.co.jp/blog/logo-design-cost/) 等のまとめ参照）。**「戦略ワークショップ＋ロゴ」**にすると倍近くになることがある。

---

### 4-2. Web サイト（MVP〜小規模店舗）

| パッケージ | ページ数目安 | 税抜の目安（円） |
|------------|--------------|------------------|
| テンプレ＋差し替え | 1〜3P | **10 万〜35 万**（内製ツール利用含む） |
| オーダーメイド UI（静的〜軽い CMS） | 3〜5P | **35 万〜90 万** |
| 撮影・コピー込み・CMS 運用研修 | 同上 | **60 万〜150 万** |

市場では Web 単体 **30 万〜300 万** と幅広く言われることが多い（[LeadGrid ブランディング費用記事](https://goleadgrid.com/blog/branding-cost) 等）。**代官山レベルの見せ方**を求めるとデザイン・写真の単価が上がりやすい。

---

### 4-3. LINE 公式（構築・初期）

| 項目 | 税抜の目安（円） |
|------|------------------|
| アカウント開設支援＋リッチメニュー＋ウェルカム＋キーワード応答 | **5 万〜20 万** |
| 上記＋配信テンプレ数本＋クーポン設計＋店頭 QR デザイン | **15 万〜40 万** |

LINE の月額・メッセージ従量は [LINE 公式の料金ページ](https://www.linebiz.com/jp/service/line-official-account/) で別途。

---

### 4-4. セットで見積もる場合（「今回のフォルダ一式」に近いスコープ）

| 組み合わせ | 税抜のざっくり幅 |
|------------|------------------|
| ブランド（ライト）＋ Web（テンプレ寄り）＋ LINE（基本） | **35 万〜80 万** |
| ブランド（スタンダード）＋ Web（オーダー MVP）＋ LINE（拡張） | **80 万〜200 万** |
| 戦略ワーク＋ブランドフル＋ Web 撮影込み＋広告クリエ | **200 万〜400 万＋** |

**あなたが受注側の場合のコツ**

- 見積は **「戦略 / ブランド / Web / LINE / 月次運用」**で行を分け、オプション化する。
- **修正回数・写真（誰が撮るか）・ドメイン・ホスティング・年間保守**を明記するとトラブルが減る。
- `docs/01`〜`04` をそのまま **スコープ定義書**として添付できる。

## 5. 非プロでも「安い起点」から交渉・進行する

プロ並みの品質は求めず、**まず出す・回す**ことを優先するときの考え方です。

### 5-1. 単価を下げる交渉の軸（外注・お願いする側）

相場を「下から」寄せるには、**作業量を減らす条件**を先に出すのが効きます。

| 条件 | 相手にとっての意味 |
|------|-------------------|
| **テンプレート指定**（Wix / STUDIO / WordPress 有料テーマ等） | デザイン・実装の初速が上がる |
| **原稿・写真はこちら支給** | ライティング・撮影の見積が消える |
| **ページは 3 枚まで**（トップ・メニュー・アクセス） | 工数の上限が明確 |
| **修正は 2 ラウンドまで** | 手戻りリスクを限定 |
| **ロゴは後回し**（いったんフォント＋店名のタイポのみ） | ブランド費をフェーズ分割 |
| **LINE は「開設＋リッチメニュー 1 枚」だけ** | 配信設計・クーポンは自分たちで後から |

伝え方の例: 「予算が限られているので、**テンプレ＋差し替え**で構いません。オリジナルデザインは不要です。」

### 5-2. 自分が請け負う場合（実務レベルを明言して低価格にする）

プロではないことを隠さず、**範囲と責任分界**を書面で揃えると、安価でも受けてもらいやすく、トラブルも減ります。

- **明言する**: ポートフォリオは別分野／学習中であること。
- **含める**: テンプレ選定、文言の入れ替え、公開までのクリック作業。
- **含めない（または別料金）**: 法務チェック、SEO 保証、複雑なカスタム、無制限修正。
- **金額の置き方**: 「材料費（ドメイン・サーバー・テーマ代）は実費、作業は ○ 万円」のように **実費と分離**すると納得されやすい。

### 5-3. ほぼ「ツール代だけ」に寄せる下限の目安

外注ゼロ〜最小にした場合の **かかるもののオーダー感**（2026 年前後の一般的なイメージ。各サービスは公式料金で確認）。

| 項目 | 目安 |
|------|------|
| **ドメイン** | 年 **約 1,000〜2,000 円台**〜 |
| **ノーコード Web**（Wix 等の有料プラン） | 月 **約 1,000〜2,000 円台**〜 |
| **ロゴ風の見出し** | Canva 無料〜 Pro（月数百円〜）／アイコン素材サイトの利用規約に注意 |
| **LINE 公式** | プランにより [LINE 公式の料金](https://www.linebiz.com/jp/service/line-official-account/)（無料枠〜） |

このルートなら **「制作費 0 + 年間 1〜3 万円程度の固定費」** で公開まで持っていくことも可能。代わりに **見た目の独自性・保守の手間**は自分側に残ります。

### 5-4. 安くしても切らさない最低ライン

- **営業時間・定休・住所・価格**は Web と Google ビジネスプロフィールで一致させる。
- **健康・効能**の表現は強く言い切らない（景表・薲機の一般知識だけでも可）。
- **スマホで読める文字サイズ**、リンク切れチェック。
