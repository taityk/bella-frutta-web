## 1. 商品・体験の「事実ベースのストーリー」（ドラフト）

会話で整理された内容。**レシピ・表示と矛盾がないか**は必ず店舗で確認してから対外文に使う（`01` のガードレール: 実物と異なる表現をしない）。

- **仕入れ**: 信頼できる農家・生産者から買い付けた果物を使う。
- **一般的なスムージーとの違い（説明用）**: 水・氷などで伸ばすタイプだと果物比率が相対的に下がりがちな一方、当店は **水・氷を使わない、果実100%** を目指している。
- **味・食感**: のど越しが **濃く・とろみがある**（ミキサーで作る様子を見せられる）。
- **透明性**: 作り方を見せることで、中身への信頼につなげる。
- **試食体験**: 必ず果物を試食してもらうことで「飲む体験」から「五感で味わう体験」へ。試食が購買と信頼につながり、リピートにつながる。
- **栄養価・無添加**: 栄養価の観点でも信頼できるものを提供。（例: いちごスムージーで子供の風邪が治ったエピソード）`[要確認]` 対外文での使用は店舗側と確認の上、薬機法・景表法に配慮した表現に調整する。
- **リピーターの多さ**: 果物の価値を体感 → 信頼・喜び → 購入 → リピート。1年を通じて果物が変わることが来店動機になっている。

### 主要仕入れ先（こだわりセクション・/calendar のコンテンツ素材）

| 仕入れ先 | 特徴 | 掲載メモ |
|----------|------|----------|
| **雪みるく農家** | 品質の高い農家。店頭価格帯が異なるケースあり（`[要確認]` Web での価格触れ方の可否を確認） | こだわりのストーリーとして紹介可能 |
| **香川 あきやま農園（デコポン）** | 2〜3ヶ月熟成してから出荷するため非常に甘い。高いテクニックが必要な栽培方法 | 産地・熟成の物語として /calendar や果物カレンダーに掲載 |
| **させほ 出島の花（みかん）** | 糖度18度（通常10度）。品種: あじっこ・なつみ。100%ジュースは即完売の実績 | 希少性・数値での説得力あり。英語表記も要検討 |

※「100％果物」など **数字や絶対表現** は、実際の配合・表示ルールと照合してから採用する。

## 2. ブランドの核（`01-brand-strategy.md` より）

**機能的価値**: フレッシュな果物をベースにした、体にうれしい一杯  
**情緒的価値**: 代官山で過ごす時間の「ご褒美」「ちょい贅沢」「自分へのケア」  
**社会的価値（任意）**: 無添加・ローカル産・フードロス削減など、**事実があるものだけ**記載

オーナーが大切にしたいこと（例・要確認）:

> 「季節の果物そのものの甘さを、足さないものは足さずに届けたい」

**差別化の方向**

1. 果物の選び方・配合（産地・季節・レシピのストーリー）— **セクション 1 とここを接続する**
2. 代官山という文脈（店の佇まい・スタッフ・一言コピーとの一貫性）
3. 見た目と写真映え（Instagram と実物のギャップを作らない）

**トーン**

- やわらかく、押し付けない。専門用語より **果物の名前と季節**
- フレンドリーだが安売り感は出さない
- 推奨語の例: 季節、フレッシュ、一杯、代官山、果実、ととのう、ご褒美
- 避けたい: 過度な「最強」「絶対」、根拠のない健康効能（景表法・薬機法）

**ガードレール（全チャネル）**

- 実物と異なる表現をしない
- 健康効能は許容範囲内に
- 景表: 最上級・No.1 等は根拠必須

## 3. メッセージング柱（`02-marketing-strategy.md` より）

コピー案を出すときは、次の **どの柱に属するか** を先に決めるとブレにくい。

1. **フレッシュ・季節** — その時期の果物の美味しさ、仕入れの短いストーリー
2. **代官山での体験** — 立ち寄りやすさ、散歩の文脈
3. **自分へのケア** — 一息・ご褒美（効能は慎重に）
4. **五感で味わう体験（唯一無二）** — 水・氷ゼロ、試食→購買、作り手が見える安心感。「飲む」ではなく「体験する」スムージー（2026年4月打ち合わせで強調）

## 4. チャネル別に「何を書くか」（`02` / `03` / `04` より）

| チャネル | 役割 | コピーの型 |
|----------|------|------------|
| **Instagram** | 世界観・新商品・日々のフック | 短いリード＋果物名・産地。作る様子は「見える化」 |
| **Web** | 公式情報の保管・検索 | ヒーロー一言、セクション見出し、果物へのこだわり・作り手・メニュー説明（`03` のセクション案に対応） |
| **LINE** | リピート・限定・臨時 | ウェルカム・キーワード応答は [`04`](./04-line-official-project.md) のドラフトをブランドトーンで調整 |
| **Google ビジネス** | 地図・時間 | 他チャネルと営業時間・住所の表記を完全一致 |

**ジャーニー**（`02`）: 認知（Instagram 等）→ 興味（**メニュー・時間・場所は Web / LINE で補完**）→ 来店 → シェア → リピート（LINE）

## 7. コピーを考えるプロセス（このプロジェクト用）

1. **事実**: セクション 1 を最新化（配合・見せ方・仕入れの言える範囲）。
2. **柱**: セクション 3 のどれか（または複合）で書くか決める。
3. **トーン**: セクション 2 の推奨／避けたいに照合。
4. **長さ**: Web ヒーローは短く。Instagram は 2〜4 行＋果物名。LINE は [`04`](./04-line-official-project.md) の分量感。
5. **検証**: ガードレール（誇大・効能・実物不一致がないか）。必要なら最終はオーナー承認。
6. **反映**: 確定文を `01` の例文・タグライン、`web` の定数、Instagram プロフィールに **同じ言葉で** 反映する。
